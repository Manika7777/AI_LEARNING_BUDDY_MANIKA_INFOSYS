# 🎓 AI Learning Buddy Manika

## 📌 Overview
AI Learning Buddy Manika is an educational web application that helps students learn concepts using Google's Gemini AI. Users can understand concepts, view real-life examples, generate quizzes, and ask any academic question.

## 🚀 Features

- Explain Concepts
- Real-Life Examples
- Generate Quiz
- Ask Anything
- Clean and Interactive Streamlit UI

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## 📂 Project Structure

```
AI_LEARNING_BUDDY/
│
├── app.py
├── requirements.txt
├── README.md
├── .env (Local Only)
└── .gitignore
```

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
